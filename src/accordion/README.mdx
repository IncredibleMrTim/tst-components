# Accordion Component

---

### CSS Class Overrides

| ClassName         | Description                                                               |
| ----------------- | ------------------------------------------------------------------------- |
| accordion-wrapper | A wrapper around the entire accordion                                     |
| accordion         | override for the actual Accordion component from @material-tailwind/react |
| accordion-heading | A wrapper around the accordion heading                                    |
| accordion-body    | A wrapper around the accordion body                                       |
| heading-wrapper   | Wraps the top level of the accordion heading                              |
| heading-left      | The left hand side of the accordion heading bar                           |
| heading-icon      | The Icon to to the left of the accordion heading bar                      |
| heading-title     | The title text within the accordion heading bar                           |
| heading-chevron   | the chevron Icon to the right of the accordion heading bar                |
